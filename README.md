#workout-visualizations

A simple Backbone Marionette application to view TrainingPeaks workout performance data.

## Requirements

Node.js and Grunt must be installed to run unit tests and serve the application. Run the usual `npm install` to set up the project.

## Running Unit Tests

Run `grunt jasmine`

## Running the Apllication

Run `grunt connect` to start a local web server. The application will be available at http://localhost:3000/index.html
